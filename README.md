# ⚾ AI Face Recognition - 棒球教練與球員辨識系統

這是一個基於 **Deep Learning (深度學習)** 技術開發的人臉辨識專案，專門用於精準辨識影片中的特定人物（例如：陳傑憲、曾豪駒）。

## 🌟 專案特色
- **高精準度**：採用 `face_recognition` 庫（基於 dlib 的 ResNet 模型），準確率高達 99.3%。
- **多照片強化**：支援每個辨識對象上傳多張照片，提升不同角度、表情的辨識能力。
- **跳幀優化**：加入跳幀處理（Frame Skipping），在不犧牲準確度的情況下大幅提升運算流暢度。
- **自動化標註**：自動在影片中框選人物並標註姓名。

## 🛠️ 環境需求
請確保你的電腦已安裝以下環境：
- Python 3.12+
- OpenCV 4.9.0.80 (穩定版)
- face_recognition
- dlib
- Numpy 1.26.4 (穩定版，避免版本衝突)
